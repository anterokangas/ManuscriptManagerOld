|           Eletään 1800-luvun puoliväliä Wienissä. Kevät            |
|           on kauneimmillaan. Aurinko paistaa kirkkaasti            |
|           pilvettömältä taivaalta. Ringstrassen kuulut             |
|           katukahvilat ovat täynnä kauniista säästä                |
|           nauttivia seurueita. Itävallan säveltäjät                |
|           luovat toinen toistaan hienompia sävellyksiä.            |
|           Erityisesti Straussien valssit, polkat ja                |
|           polkka-masurkat ovat valloittaneet niin                  |
|           Itävallan kuin koko muun Euroopan.                       |
|                                                                    |
|           Musiikin ja tanssiaisten suuret ystävät,                 |
|           herrat Franz ja Josef, istuvat kahvilassa ja             |
|           ovat mielipuuhassan, he nimittäin kiistelevät,           |
|           kuinka ja missä polkka on syntynyt.                      |
|                                                                    |
|      I KOHTAUS: KAHVILASSA                                         |
|                                                                    |
|      Herrat Franz ja Josef ovat kävelyllä Wienin                   |
|      Ringstrassella. Kahvilasta kuuluu valssimusikkia. He          |
|      pysähtyvät.                                                   |
|                                                                    |
|                          FRANZ                                     |
|                Tuohan kuulostaa kuin Johan Straussin, niiden       |
|                kolmen kuuluisan säveltäjän isän,                   |
|                valssimusiikilta. Eiköhän poiketa kahville          |
|                kuuntelemaan ja nauttimaan siitä.                   |
|                                                                    |
|                          JOSEF                                     |
|                No niinpäs kuulostaakin. Mennään vain.              |
|                                                                    |
|                          FRANZ                                     |
|                Taisimme erehtyä, musiikki onkin Pas                |
|                d'Espagnea ja täällä alkavat ne kuuluisat           |
|                Tanssiaiset.                                        |
|                                                                    |
|                          JOSEF                                     |
|                Ei se mitään. Mutta emme taida ehtiä vielä          |
|                juoda kahvia, sillä meidän pitää hakea kauniit      |
|                daamimme ja lähteä tanssimaan.                      |
|                                                                    |
|                          FRANZ                                     |
|                Niinpä teemmekin. Hei, tulkaa kaikki                |
|                tanssimaan.                                         |
|                                                                    |
|      II KOHTAUS: KONSERTISSA                                       |
|                                                                    |
|      III KOHTAUS: KESÄJUHLISSA                                     |
|                                                                    |
|      Franz ja Josef on kutsuttu kassakaappifirman                  |
|      kesäjuhliin.                                                  |
|                                                                    |
|                          JOSEF                                     |
|                Wertheim-yhtiö kertoo myyneensä 20 000:nnen         |
|                kassakaappinsa.                                     |
|                                                                    |
|                          FRANZ                                     |
|                Uskon kyllä, nehän ovat hyvin luotettavia ja        |
|                kestäviä. Hehän järjestivät sen esityksenkin,       |
|                jossa kaappia pidettiin tulessa melko pitkään.      |
|                Kaappi ei mennyt rikki eivätkä sisällä olleet       |
|                paperitkaan olleet palaneet.                        |
|                                                                    |
|                          JOSEF                                     |
|                Ei siis mikään ihme, että he mainostavat            |
|                tuotettaan tukenkestäväksi.                         |
|                                                                    |
|                          FRANZ                                     |
|                Juuri siksi meidänkin yhtiömme luottaa heidän       |
|                kaappeihinsa.                                       |
